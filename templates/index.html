Rent a thing API

API created as a simulation of some traffic in my mobile app.


Docs

List of examples you can use

GET /users - Get list of all registered users

GET /offers - Get list of all created offers

GET /users/1 - Get a single user with id 1

GET /offers/7 - Get a single offer with id 7

GET /offers?userId=3 - Filter offers by user id 3
